**Live URL:** https://interiorsdegrai.in  
**Hosted on:** GitHub Pages

## Pages
- `index.html` — Home
- `about.html` — About Us
- `services.html` — Services
- `projects.html` — Projects Portfolio
- `contact.html` — Contact & Estimate

## How to Deploy to GitHub Pages
1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Settings → Pages
4. Set Source to "Deploy from branch: main"
5. Your site will be live in a few minutes!

## Tech Stack
- Pure HTML5, CSS3, JavaScript (no frameworks)
- Font Awesome 6.5 (CDN)
- Google Fonts: Jost (fallback for Futura)
- All CSS & JS embedded inline (self-contained pages)
- Logo embedded as base64 PNG
