## GitHub Pages Hosting
1. Create new GitHub repo
2. Upload all files (maintain folder structure: `css/` and `js/`)
3. Settings → Pages → Deploy from branch → main → / (root)
4. Site goes live at `https://yourusername.github.io/degrai-interiors/`
